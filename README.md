# Cats Fan Page

A simple, responsive fan page celebrating cats. Includes a gallery, fun facts, and a dark mode toggle.

## How to use

- Open `index.html` in your browser (double-click it).
- Click "Show a random cat fact" for a new fact.
- Toggle the moon/sun button to switch themes. Your choice is remembered.

## Customize

- Replace the `placekitten.com` image URLs in `index.html` with your own cat photos.
- Update colors in `styles.css` by editing the CSS variables under `:root` and `[data-theme="light"]`.
- Add more facts in `script.js` by extending the `facts` array.
